Html

(voltar)

HTML, ou Hyper Text Markup Language, é a linguagem de marcação usada para criar páginas da web. Com HTML, os desenvolvedores podem estruturar o conteúdo de uma página e adicionar elementos como texto, imagens, links e formulários, tornando-as interativas e visualmente atrativas. Além disso, o HTML permite a criação de layouts flexíveis e responsivos, que se adaptam a diferentes dispositivos, como computadores, tablets e smartphones. Dominar HTML é fundamental para quem deseja criar presença online, seja por meio de um site pessoal, blog ou loja virtual. Aprender HTML abre as portas para uma variedade de oportunidades no campo do desenvolvimento web.

Head

(voltar)

O 'head' é como a cabeça de uma página HTML, onde colocamos informações importantes para os bastidores. É onde incluímos coisas como o título da página, que aparece na aba do navegador, e outros detalhes que os usuários não veem diretamente, mas que ajudam os motores de busca a entenderem o conteúdo da página.

Já o 'title' é onde você define o título da sua página, que é exibido na aba do navegador. É como dar um nome ao seu documento HTML. Um título bacana e informativo não só ajuda os usuários a saberem do que se trata a página, mas também ajuda o Google a entender e classificar melhor seu site nos resultados de pesquisa. Em resumo, o 'head' e 'title' são como os bastidores de uma peça de teatro - eles podem não ser vistos, mas são essencia

Body

(voltar)

O 'body' é a parte de uma página HTML que contém todo o conteúdo visível para os usuários. É como o palco onde todo o drama acontece! Aqui é onde você coloca textos, imagens, vídeos, links e qualquer outra coisa que os visitantes do seu site verão e interagirão. Pense nele como o corpo principal da página.

Dentro do 'body', você pode usar diferentes elementos HTML para organizar e estruturar seu conteúdo, como 'header' para o cabeçalho, 'nav' para a navegação, 'main' para o conteúdo principal. Cada parte do 'body' contribui para a experiência do usuário e a aparência geral do seu site.

Strong

(voltar)

O elemento 'strong' é uma tag HTML usada para destacar texto importante dentro de um documento. Quando você envolve um texto com 'strong', você está dizendo aos navegadores e mecanismos de pesquisa que aquele texto é particularmente relevante ou enfático. Geralmente, o texto dentro de 'strong' é exibido em negrito por padrão nos navegadores, mas a semântica por trás disso é mais importante do que o estilo visual.

Input

(voltar)

O elemento 'input' é uma tag HTML usada para criar áreas interativas em formulários, onde os usuários podem inserir dados. Essas áreas podem variar desde campos de texto simples, onde os usuários podem digitar informações, até caixas de seleção, botões de rádio e botões de envio.

Atributos

(voltar)

Os atributos id e class são usados em HTML para identificar e estilizar elementos de forma única ou em grupos.

O atributo id é um identificador único para um elemento HTML. Cada id em um documento HTML deve ser exclusivo. Ele é frequentemente usado para identificar um elemento específico para estilização ou manipulação com JavaScript. O atributo class é usado para agrupar elementos que compartilham características semelhantes. Você pode aplicar a mesma classe a vários elementos, permitindo estilização ou manipulação em massa. Diferentemente do id, o mesmo valor de class pode ser aplicado a múltiplos elementos.

Listas

(voltar)

A tag 'ol' é usada para criar listas ordenadas, onde os itens são numerados sequencialmente, geralmente começando em 1 por padrão. Cada item da lista é definido usando a tag 'li'

Por outro lado, a tag 'ul' é usada para criar listas não ordenadas, onde os itens não são numerados sequencialmente. Em vez disso, eles são geralmente marcados com marcadores, como pontos ou círculos. Assim como 'ol', os itens de uma lista 'ul' são definidos usando a tag 'li'.

Hiperlink

(voltar)

Um hiperlink, também conhecido como simplesmente "link", é um elemento em uma página da web que permite aos usuários navegar de uma página para outra simplesmente clicando nele. Em HTML, os hiperlinks são criados usando a tag 'a' (anchor). Um hiperlink geralmente consiste em duas partes principais: o texto âncora, que é o texto visível do link, e o URL de destino, para onde o link aponta.
